# 商品计价器 Android App

一个简单易用的商品计价Android应用，支持添加商品、设置图片、计算总价和管理订单。

## 功能特点

- ✅ 添加商品（名称、价格、图片）
- ✅ 图片选择和显示
- ✅ 实时价格计算
- ✅ 订单管理和备注
- ✅ 历史订单查看
- ✅ Material Design界面

## 下载APK

1. 进入 [Actions](../../actions) 页面
2. 点击最新的构建
3. 下载 `app-debug` 文件
4. 解压后安装APK到手机

## 技术栈

- Kotlin
- Room数据库
- MVVM架构
- Material Design
- Glide图片加载

## 构建说明

项目会通过GitHub Actions自动构建APK文件。每次推送代码后都会触发自动构建。
